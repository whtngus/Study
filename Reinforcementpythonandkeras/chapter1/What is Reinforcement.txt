강화학습의 개념

스키너의 강화 연구 (스키너라는 행동 심리학자 가 처음 개념을 제시)
Reinforcement : Trial and Error(시행착오)를 통해 학습하는 방법 중 하나
정답이 주어진 것은 아니지만 그저 주어진 데이터에 대해 학습하는 것도 아님 
--> 보상을 통해서 학습(컴퓨터가 선택한 행동(Action)에 대한 환경의 반응)
장점 : 사전지식 없이 학습 가능

Agent : 스스로 학습을 하는 컴퓨터


- 순차적 행동 결정 문제의 구성 요소
1. State : Agent가 움직이는 속도와 같은 동적인 요소, Agent가 상태를 통해 행동을 결정하기에 충분한 정보를 제공해야 함
2. Action : Agent가 취할 수 있는 행동 
3. reward : 학습을 할 수 있는 핵심적인 정보
4. policy : 특정 상태가아닌 모등상태에 대해 Agent가 어떤 action을 할지 정해놓은 정책
제일 좋은정책을 최적 정책 이라함(optimal policy) - 최적 정책에 따라 행동했을 때 보상의 합을 최대로 받을 수 있음

강화학습을 하기 위해 인공신경망(Artificial Neural Network)을 사용해야 함 


브레이크아웃의 MDP와 학습 방법 (공튕겨서 벽돌깨기 게임)
1.MDP
상태 : 게임 화면 --> 4개의 화면을 연속으로 1개로 받아들임 (2차원 픽셀 데이터)
행동 : 제자리, 왼쪽,오른쪽,발사 이고 브레이크 아웃에는 발사는 게임을 시작할때 사용 
	에이전트가 행동을하면 4개의 화면을 받아올때까지 움직임을 멈춤 
보상 : 벽돌이 하나씩 꺠질 때마다 보상 +1 목숨을 잃을 경우 보상 -1
2. 학습
Q Function 을 통해 행동이 얼마나 좋은지 저장
이 양이 많아지면서 불가능한걸 DQN(Deep Q-Network)를 통해 해결
